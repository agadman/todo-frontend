# Todo App – React + CRUD API
Detta repo innehåller en Todo-applikation byggd i React och TypeScript som kommunicerar med ett eget backend-API med stöd för CRUD-operationer.

Användaren kan skapa, visa, uppdatera status och ta bort todos.
Frontend hämtar och uppdaterar data dynamiskt via API-anrop.

## Publicerad lösning
Frontend: https://prismatic-fudge-af90ec.netlify.app
API: https://todo-api-8fuh.onrender.com/api/todos

## Tekniker
	•	React + TypeScript
	•	Vite
	•	Fetch API
	•	Express (backend)
	•	PostgreSQL
	•	Deployment: Render (backend) + Netlify (frontend)

## Funktioner
	•	Hämta todos 
	•	Skapa ny todo via formulär
	•	Uppdatera status direkt i listan
	•	Ta bort todo

## Extra
	•	Validering med Yup
	•	Loading-indikator med react-spinners
